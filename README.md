# LocalBuzz ドキュメント

LocalBuzzプラットフォームのドキュメント集です。

## ドキュメント一覧

### [ユーザーガイド](./USER_GUIDE.md)

LocalBuzzを利用するユーザー向けのガイドです。以下の内容を含みます：

- ダッシュボードの使い方
- 店舗管理（店舗情報編集、Google投稿管理、GBP連携設定、業態設定）
- アンケート管理（作成、一覧、集計結果）
- 口コミ管理（一覧、返信作成、分析レポート）
- 順位計測（順位設定、履歴グラフ）
- サブアカウント管理
- LINE Bot設定
- よくある質問

**対象読者**: LocalBuzzを利用する店舗管理者、スタッフ

---

### [管理者ガイド](./ADMIN_GUIDE.md)

LocalBuzzシステムを管理する管理者向けのガイドです。以下の内容を含みます：

- ダッシュボード（システム全体の統計）
- アカウント管理（作成、編集、削除、プラン管理）
- システム管理（システム設定、通知設定、API設定）
- 業態管理設定
- 上位カテゴリ管理
- よくある質問

**対象読者**: LocalBuzzシステムの管理者、運用担当者

---

## ドキュメントの使い方

### ユーザーガイドを読む場合

1. 初めてLocalBuzzを使用する場合は、[ユーザーガイド](./USER_GUIDE.md)の「はじめに」から読み始めてください
2. 特定の機能について知りたい場合は、目次から該当するセクションに直接移動してください
3. 問題が発生した場合は、「よくある質問」セクションを確認してください

### 管理者ガイドを読む場合

1. システム管理者として初めてLocalBuzzを管理する場合は、[管理者ガイド](./ADMIN_GUIDE.md)の「はじめに」から読み始めてください
2. 特定の管理機能について知りたい場合は、目次から該当するセクションに直接移動してください
3. システム設定を変更する場合は、該当するセクションの「注意事項」を必ず確認してください

---
